# How to Connect from Xbox & Nintendo Switch

Your server IP: `98.95.134.206`
Bedrock Port: `19132`

Since Xbox and Switch don't have an "Add Server" button, we use a DNS trick
called BedrockConnect to redirect you to our server.

## Option 1: Use Public BedrockConnect DNS (Easiest)

### Xbox Setup
1. Go to **Settings → Network → Advanced Settings → DNS Settings**
2. Select **Manual**
3. Set Primary DNS to: `104.238.130.180`
4. Set Secondary DNS to: `8.8.8.8`
5. Restart Minecraft
6. Go to **Servers** tab and click on any featured server (e.g. "The Hive")
7. A "Connect to a Server" screen will appear
8. Enter IP: `98.95.134.206` and Port: `19132`
9. Click **Submit** — you're in!

### Nintendo Switch Setup
1. Go to **System Settings → Internet → Your WiFi Network → Change Settings**
2. Set **DNS Settings** to **Manual**
3. Set Primary DNS to: `104.238.130.180`
4. Set Secondary DNS to: `8.8.8.8`
5. Save and restart Minecraft
6. Go to **Servers** tab and click on any featured server
7. Enter IP: `98.95.134.206` and Port: `19132`
8. Click **Submit**

## Option 2: Use Our Self-Hosted BedrockConnect DNS

If the public DNS is slow or down, we run our own on the server.

- Primary DNS: `98.95.134.206`
- Secondary DNS: `8.8.8.8`

Follow the same steps above but use our server IP as the Primary DNS instead.

## Troubleshooting

- **"Unable to connect to world"**: The server may still be starting up. Wait 5 minutes and try again.
- **DNS not working**: Try switching between Option 1 and Option 2.
- **Lag or timeout**: Make sure your console is on a stable WiFi connection.
- **To revert DNS**: Set DNS back to **Automatic** in your network settings.
